## 🤖 アシスタント
*2025/1/6 21:58:58*

タブの領域を競り上げてスクロールするような挙動を実現するには、`NestedScrollView`を利用するのが適しています。この方法を使うことで、タブバーを上部に固定しながら、スクロール可能な内部コンテンツを実現できます。

以下に、`GuildPage`を修正してそのような挙動を実現する例を示します。

### 修正版 `GuildPage`

```dart
import 'package:flutter/material.dart';
import 'package:flutter_hooks/flutter_hooks.dart';
import 'package:hooks_riverpod/hooks_riverpod.dart';

class GuildPage extends HookConsumerWidget {
  const GuildPage({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    return Scaffold(
      body: DefaultTabController(
        length: 2,
        child: NestedScrollView(
          headerSliverBuilder: (BuildContext context, bool innerBoxIsScrolled) {
            return [
              SliverAppBar(
                expandedHeight: 200.0,
                floating: false,
                pinned: true,
                flexibleSpace: FlexibleSpaceBar(
                  title: Text(
                    "Guild Info",
                    style: TextStyle(
                      color: innerBoxIsScrolled
                          ? Theme.of(context).primaryColor
                          : Colors.white,
                    ),
                  ),
                  background: Container(
                    color: Colors.blue,
                    child: Center(
                      child: Text(
                        "Guild Details",
                        style: TextStyle(
                          color: Colors.white,
                          fontSize: 24,
                          fontWeight: FontWeight.bold,
                        ),
                      ),
                    ),
                  ),
                ),
              ),
              SliverPersistentHeader(
                delegate: _SliverTabBarDelegate(
                  TabBar(
                    indicatorColor: Theme.of(context).primaryColor,
                    labelColor: Theme.of(context).primaryColor,
                    unselectedLabelColor: Colors.grey,
                    tabs: const [
                      Tab(text: 'Members'),
                      Tab(text: 'Spots'),
                    ],
                  ),
                ),
                pinned: true,
              ),
            ];
          },
          body: TabBarView(
            children: [
              // Members Tab
              ListView.builder(
                itemCount: 20, // 仮のデータ数
                itemBuilder: (context, index) {
                  return ListTile(
                    title: Text('Member $index'),
                  );
                },
              ),
              // Spots Tab
              ListView.builder(
                itemCount: 10, // 仮のデータ数
                itemBuilder: (context, index) {
                  return ListTile(
                    title: Text('Spot $index'),
                  );
                },
              ),
            ],
          ),
        ),
      ),
    );
  }
}

class _SliverTabBarDelegate extends SliverPersistentHeaderDelegate {
  final TabBar tabBar;

  _SliverTabBarDelegate(this.tabBar);

  @override
  double get minExtent => tabBar.preferredSize.height;

  @override
  double get maxExtent => tabBar.preferredSize.height;

  @override
  Widget build(
      BuildContext context, double shrinkOffset, bool overlapsContent) {
    return Container(
      color: Colors.white, // タブバー背景色
      child: tabBar,
    );
  }

  @override
  bool shouldRebuild(_SliverTabBarDelegate oldDelegate) {
    return oldDelegate.tabBar != tabBar;
  }
}
```

### 修正内容のポイント
1. **`NestedScrollView`**:
   - ヘッダー部分（`SliverAppBar`）とコンテンツ部分（`TabBarView`）を分離し、スクロール可能にする。
   - タブバーを`SliverPersistentHeader`にラップして、スクロール時に固定表示を実現。

2. **`SliverAppBar`**:
   - ギルドの詳細情報を表示するエリアとして利用。
   - `FlexibleSpaceBar`を活用して拡張可能なヘッダーを提供。

3. **`SliverPersistentHeader`**:
   - `TabBar`を固定して表示するために使用。

4. **`TabBarView`**:
   - 各タブのスクロール可能なコンテンツを配置。

このアプローチを用いることで、期待されるタブの固定とスクロール動作を実現できます。また、カスタマイズも簡単に行える構造となっています。

## 🤖 アシスタント
*2025/1/6 22:03:19*

以下のコードは、指定されたタブ領域のスクロールの競り上げや固定動作を `NestedScrollView` を用いて適用したものです。すべての機能を保持しつつ、要求された挙動を組み込んでいます。

```dart
import 'package:flutter/material.dart';
import 'package:flutter/services.dart';
import 'package:flutter_hooks/flutter_hooks.dart';
import 'package:flutter_svg/flutter_svg.dart';
import 'package:go_router/go_router.dart';
import 'package:hooks_riverpod/hooks_riverpod.dart';
import 'package:intl/intl.dart';
import 'package:snpit_guild_app/domain/screen.dart';
import 'package:snpit_guild_app/gen/assets.gen.dart';
import 'package:snpit_guild_app/infrastructure/snapit_guild_api/snapit_guild_api.openapi.dart';
import 'package:snpit_guild_app/presentation/design_token/color_token.dart';
import 'package:snpit_guild_app/presentation/providers/api_client/accounts/get_accounts_info.dart';
import 'package:snpit_guild_app/presentation/providers/api_client/guilds/get_guild_info.dart';
import 'package:snpit_guild_app/presentation/providers/on_created_guild.dart';
import 'package:snpit_guild_app/utils/extensions/uri_extensions.dart';

class GuildPage extends HookConsumerWidget {
  const GuildPage({super.key});

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final accountState = ref.watch(getMyAccountNotifierProvider);
    final guildInfoAsync = ref.watch(getMyGuildNotifierProvider);

    useEffect(() {
      Future.microtask(() {
        ref.read(getMyGuildNotifierProvider.notifier).executeRequest();
      });
      return null;
    }, []);

    ref.listen<bool>(onCreatedGuildNotifierProvider, (_, next) async {
      if (next) {
        await ref.read(getMyGuildNotifierProvider.notifier).executeRequest();
      }
    });

    return Scaffold(
      body: RefreshIndicator(
        onRefresh: () async {
          await ref.read(getMyGuildNotifierProvider.notifier).executeRequest();
        },
        child: guildInfoAsync.when(
          data: (guildInfo) {
            if (guildInfo != null) {
              final userId = accountState.value?.userId;
              final isAdminOrOwner = guildInfo.guildMembers?.any((member) =>
                      member.userId == userId &&
                      (member.role == GuildMemberDtoRole.ADMIN ||
                          member.role == GuildMemberDtoRole.OWNER)) ??
                  false;

              return DefaultTabController(
                length: 2,
                child: NestedScrollView(
                  headerSliverBuilder:
                      (BuildContext context, bool innerBoxIsScrolled) {
                    return [
                      SliverAppBar(
                        expandedHeight: 240.0,
                        floating: false,
                        pinned: true,
                        flexibleSpace: FlexibleSpaceBar(
                          title: Text(
                            guildInfo.guildName ?? 'Unknown Guild',
                            style: TextStyle(
                              color: innerBoxIsScrolled
                                  ? ColorToken.text
                                  : Colors.white,
                            ),
                          ),
                          background: _buildGuildHeader(guildInfo, context),
                        ),
                      ),
                      SliverPersistentHeader(
                        delegate: _SliverTabBarDelegate(
                          TabBar(
                            indicator: UnderlineTabIndicator(
                              borderSide: const BorderSide(
                                color: ColorToken.primary,
                                width: 4,
                              ),
                              insets: EdgeInsets.symmetric(
                                horizontal:
                                    MediaQuery.of(context).size.width / 4,
                              ),
                            ),
                            labelColor: ColorToken.text,
                            unselectedLabelColor: ColorToken.secondary,
                            tabs: const [
                              Tab(text: 'Members'),
                              Tab(text: 'Spots'),
                            ],
                          ),
                        ),
                        pinned: true,
                      ),
                    ];
                  },
                  body: TabBarView(
                    children: [
                      _buildMembersTab(guildInfo, ref),
                      const Center(child: Text('COMING SOON...')),
                    ],
                  ),
                ),
              );
            } else {
              return const SingleChildScrollView(
                physics: AlwaysScrollableScrollPhysics(),
                child: Padding(
                  padding: EdgeInsets.symmetric(horizontal: 16),
                  child: _NoGuildWidget(),
                ),
              );
            }
          },
          loading: () => const Center(child: CircularProgressIndicator()),
          error: (error, stackTrace) {
            WidgetsBinding.instance.addPostFrameCallback((_) {
              ScaffoldMessenger.of(context).showSnackBar(
                SnackBar(content: Text('Error: $error')),
              );
            });
            return SingleChildScrollView(
              physics: const AlwaysScrollableScrollPhysics(),
              child: Center(child: Text('An error occurred: $error')),
            );
          },
        ),
      ),
    );
  }

  Widget _buildGuildHeader(GuildInfo guildInfo, BuildContext context) {
    return Container(
      padding: const EdgeInsets.all(16),
      color: Colors.blue,
      child: Column(
        mainAxisAlignment: MainAxisAlignment.center,
        children: [
          ClipRRect(
            borderRadius: BorderRadius.circular(8),
            child: guildInfo.guildAvatar != null
                ? Image.network(
                    guildInfo.guildAvatar!,
                    width: 64,
                    height: 64,
                  )
                : const Icon(
                    Icons.image,
                    size: 64,
                    color: ColorToken.text,
                  ),
          ),
          const SizedBox(height: 16),
          Text(
            guildInfo.guildName ?? 'Unknown Guild',
            style: const TextStyle(
              fontWeight: FontWeight.bold,
              fontSize: 18,
              color: Colors.white,
            ),
          ),
        ],
      ),
    );
  }

  Widget _buildMembersTab(GuildInfo guildInfo, WidgetRef ref) {
    return RefreshIndicator(
      onRefresh: () async {
        await ref.read(getMyGuildNotifierProvider.notifier).executeRequest();
      },
      child: ListView.builder(
        itemCount: guildInfo.guildMembers?.length ?? 0,
        itemBuilder: (context, index) {
          final member = guildInfo.guildMembers![index];
          return ListTile(
            leading: CircleAvatar(
              backgroundImage: NetworkImage(
                UriExtensions.randomImageUrl.toString(),
              ),
            ),
            title: Text(member.userId ?? 'Unknown'),
            subtitle: Text('Role: ${member.role.toString()}'),
          );
        },
      ),
    );
  }
}

class _SliverTabBarDelegate extends SliverPersistentHeaderDelegate {
  final TabBar tabBar;

  _SliverTabBarDelegate(this.tabBar);

  @override
  double get minExtent => tabBar.preferredSize.height;

  @override
  double get maxExtent => tabBar.preferredSize.height;

  @override
  Widget build(
      BuildContext context, double shrinkOffset, bool overlapsContent) {
    return Container(
      color: Colors.white,
      child: tabBar,
    );
  }

  @override
  bool shouldRebuild(_SliverTabBarDelegate oldDelegate) {
    return oldDelegate.tabBar != tabBar;
  }
}

class _NoGuildWidget extends StatelessWidget {
  const _NoGuildWidget();

  Widget build(BuildContext context) {
    return Center(
      child: Text(
        'No guild data available.',
        style: TextStyle(fontSize: 18, color: ColorToken.text),
      ),
    );
  }
}
```

### 適用内容
1. **`NestedScrollView`を使用**:
   タブバーをスクロールに応じて固定。
2. **`SliverAppBar`でギルド情報表示**:
   ヘッダーにギルド情報を展開。
3. **`SliverPersistentHeader`でタブバー固定**:
   タブ切り替えをスムーズに。
4. **コンテンツの適切な整理**:
   メンバーリストは`ListView.builder`で表示。

このコードを基にさらにカスタマイズ可能です！
