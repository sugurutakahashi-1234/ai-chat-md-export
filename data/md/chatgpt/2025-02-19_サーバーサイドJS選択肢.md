### 1. \*\*Node.js\*\*
   - 最も一般的なサーバーサイド JavaScript ランタイム
   - \*\*特徴\*\*
     - 非同期 I/O による高パフォーマンス
     - V8 エンジン上で動作
     - 豊富な npm パッケージ
   - \*\*適用例\*\*
     - REST API
     - WebSocket サーバー
     - バックエンド開発全般

### 2. \*\*Deno\*\*
   - Node.js の作者が新たに開発したセキュリティ重視の JavaScript/Type
