## 🤖 アシスタント
*2025/1/6 14:07:40*

Reactフックは、関数コンポーネントで状態やライフサイクルを扱うための機能です。

主要なフック：

1. **useState** - 状態管理
```jsx
const [count, setCount] = useState(0);
```

2. **useEffect** - 副作用の処理
```jsx
useEffect(() => {
  console.log('Component mounted');
  return () => console.log('Cleanup');
}, []);
```

3. **useContext** - コンテキストの利用
4. **useReducer** - 複雑な状態管理
5. **useMemo** - 計算結果のメモ化
6. **useCallback** - 関数のメモ化
